# Setup

App uses yarn with pnp

To launch application:

1. Fetch packages `yarn`
2. Install sdk for your ide (for vscode `yarn dlx @yarnpkg/sdks vscode`)
3. Start application `yarn start`

# How to use

1. `/login` - Login page
2. `/table` - Main table
